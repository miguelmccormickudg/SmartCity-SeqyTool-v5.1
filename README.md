# SmartCity-SeqyTool-v5.1
SmartCity SeqyTool v5.1 (FactaTerrae) es un modelo digital interactivo que evalúa la madurez territorial de municipios y regiones. Usa geografía digital y ciencia de datos para analizar indicadores clave (gobernanza, movilidad, infraestructura) y generar dashboards dinámicos de diagnóstico y planificación. Herramienta abierta y sostenible.
